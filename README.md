# 🌱 GMATE Admin Dashboard

A modern, scalable **Admin Dashboard** built to manage a smart rooftop planting platform.  
It provides full control over users, system data, and communication through a clean and intuitive interface.

---

## 🚀 Overview

GMATE Admin Dashboard is designed as a centralized control panel for managing a platform that connects users with smart task management solutions.

It focuses on:
- Efficient user management  
- Real-time platform insights  
- Seamless communication handling  
- Clean and responsive UI/UX  

---

## ✨ Features

### 👥 User Management
- View all users with detailed information  
- Role-based access (Admin / User)  
- Activate / Deactivate users  
- Edit and delete accounts  
- Search and filter users  
- Pagination for scalability  

---

### 📊 Dashboard Analytics
- Total users / active users  
- Total projects  
- Messages overview  
- Quick actions for faster workflow  

---

### 💬 Contact Messages System
- View all incoming messages  
- Mark messages as read/unread  
- Delete messages  
- Search within messages  

---

### ⚡ Performance & UX
- Optimized rendering for large datasets  
- Loading states for better UX  
- Clean dark-themed modern UI  
- Fully responsive design  

---

## 🛠️ Tech Stack

- **Frontend:** Next.js  
- **Styling:** Tailwind CSS  
- **State Management:** Zustand 
- **API Integration:** Axios
- **Deployment:** Vercel  

---

## 🧠 Architecture Highlights

- Component-based scalable structure  
- Separation of concerns (UI / Logic / API)  
- Reusable components (cards, tables, actions)  
- Clean folder organization for maintainability  

---

## 🔐 Security & Access Control

- Role-based access system (RBAC)  
- Protected routes  
- Controlled admin actions
