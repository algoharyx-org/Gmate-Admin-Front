## ✨ Overview

**Gmate** is a high-performance, AI-driven platform designed for career management and productivity. This "Prime Version" marks a total architectural shift, moving from legacy Context API to a high-speed **Zustand** store and a futuristic **VisionOS-inspired UI**.

---

## 🛠️ Tech Stack (The Prime Source)

| Category             | Technology                                                                 |
| -------------------- | -------------------------------------------------------------------------- |
| **Frontend** | **React 19** (Latest), Vite 7                                              |
| **Styling** | **Tailwind CSS v4**, Glassmorphism, Framer Motion                          |
| **State Management** | **Zustand** (useTaskStore)                                                 |
| **Data Fetching** | **TanStack Query v5** |
| **Icons & UI** | Lucide React, Radix UI (Shadcn/UI Custom)                                  |
| **Dev Environment** | **Arch Linux**, **Ghostty Terminal**, **Neovim** |

---

## 🚀 Key Features

* **💎 VisionOS Glassmorphism**: A stunning, semi-transparent UI with mesh backgrounds and high-end blur effects.
* **🤖 AIAssistant FAB**: A floating smart assistant integrated into the dashboard for real-time help.
* **📋 Pro Kanban Board**: Advanced task management powered by Zustand for zero-latency updates.
* **⚡ Prime Architecture**: Unified project structure following Clean Architecture principles.
* **📊 Dynamic Dashboards**: Real-time project tracking and performance stats.
